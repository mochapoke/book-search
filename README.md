# 개요

- 독서 앱
- 책 검색을 할 수 있다.
- 책에 별점을 매길 수 있다.
- 책에 한줄평을 남길 수 있다.
- 책의 주요 디테일을 확인할 수 있다.
- 네이버 api를 활용하여 책 검색 웹앱을 만든다
  [네이버 api](https://developers.naver.com/docs/search/book/)

## 기간

2020.04 ~ 2020.04
